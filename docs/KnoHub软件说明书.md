**版本：** V1.0  
**日期：** 2026-01-01

## 1. 软件简介

### 1.1 软件概述
KnoHub 是一款面向校园场景的一站式知识库管理系统。它采用现代化的前后端分离架构，旨在为高校师生提供高效、便捷的课程资料、技术文档及校园信息管理服务。系统通过直观的卡片式设计和强大的文件管理功能，解决了传统校园资源存储分散、检索困难、格式不兼容等痛点。

### 1.2 主要功能
*   **多维度资源管理**：支持课程资料、技术文档、校园信息三大模块分类存储。
*   **可视化文件组织**：提供树形目录结构，支持文件夹创建、文件重命名及直观的拖拽排序功能。
*   **强大的在线预览**：
    *   **办公文档**：支持 PDF、DOC/DOCX 格式文档的无需下载直接预览。
    *   **专业格式**：独家支持 Logisim (.circ) 电路设计文件的可视化渲染预览，方便计算机专业教学。
*   **数据看板**：实时展示系统活跃用户数与最近更新资源。

## 2. 运行环境

### 2.1 硬件环境
*   **服务器**：
    *   CPU：2核心及以上
    *   内存：4GB及以上
    *   硬盘：50GB及以上可用空间
*   **客户端**：
    *   支持现代主流浏览器（Chrome, Firefox, Edge, Safari）的 PC 或笔记本电脑。

### 2.2 软件环境
*   **操作系统**：Linux (Ubuntu/CentOS 推荐), Windows Server, 或 macOS。
*   **运行依赖**：
    *   JDK 17 或更高版本
    *   Node.js 18+ (用于前端构建)
    *   PostgreSQL 14+ (生产环境推荐) 或 H2 Database (开发/测试)
    *   Docker (可选，用于快速部署数据库)

## 3. 软件安装与配置

### 3.1 数据库准备
1.  **开发模式**：系统默认使用内置 H2 数据库，无需额外安装，直接启动即可。
2.  **生产模式**：推荐安装 PostgreSQL。
    *   使用 Docker 快速启动：
        ```bash
        sudo ./scripts/postgres_up.sh
        source ./scripts/postgres_env.sh
        ```
    *   或手动配置环境变量（`DB_URL`, `DB_USERNAME`, `DB_PASSWORD`）。

### 3.2 后端部署
1.  进入 `backend` 目录。
2.  执行构建命令：
    ```bash
    mvn clean package -DskipTests
    ```
3.  启动服务：
    ```bash
    java -jar target/backend-0.0.1-SNAPSHOT.jar
    ```
    后端服务默认运行在 `8080` 端口。

### 3.3 前端部署
1.  进入 `frontend` 目录。
2.  安装依赖并构建：
    ```bash
    pnpm install
    pnpm build
    ```
3.  将 `dist/` 目录下的静态文件部署至 Nginx 或其他 Web 服务器。

## 4. 软件功能与操作使用

### 4.1 首页与概览
用户访问系统首页，首先看到的是简洁的仪表盘。
*   **活跃用户统计**：顶部实时显示当前系统的活跃访客数量。
*   **最近更新**：展示最近上传或修改的资源卡片，方便快速访问热点内容。
*   **导航栏**：顶部导航栏提供 "课程资料"、"技术文档"、"校园信息" 的快速入口。

> （此处建议插入：首页界面截图）

### 4.2 资源管理
系统以“资源卡片”为核心进行管理。
1.  **创建资源**：点击“新建资源”按钮，填写资源名称、描述，并选择所属分类（如课程资料）。
2.  **标签管理**：支持为资源添加自定义标签（Tag），便于后续检索。
3.  **编辑与删除**：管理员可对已有资源卡片进行元数据修改或删除操作。

> （此处建议插入：资源创建/编辑弹窗截图）

### 4.3 文件与目录管理
点击任一资源卡片，进入详细的文件管理界面。
*   **树形目录**：左侧展示清晰的文件树结构。
*   **文件夹操作**：支持新建文件夹、无限层级嵌套。
*   **文件上传**：
    *   支持点击上传或直接拖拽文件到指定区域。
    *   支持批量文件上传。
*   **拖拽排序**：用户可以通过鼠标长按并拖拽文件或文件夹，自由调整它们在列表中的顺序。
*   **重命名与删除**：右键或点击操作菜单即可重命名或软删除文件。

> （此处建议插入：文件树及拖拽操作截图）

### 4.4 智能文档预览
KnoHub 提供了强大的无需下载预览功能，极大提升了资料查阅效率。

#### 4.4.1 PDF 与 Word 预览
点击文件列表中的 PDF 或 DOC/DOCX 文件，系统会在右侧预览区域直接渲染文档内容。
*   **PDF**：支持翻页、缩放、目录跳转。
*   **Word**：自动转换为 HTML 格式展示，保留原有排版。

> （此处建议插入：PDF/Word 预览界面截图）

#### 4.4.2 Logisim 电路图预览
针对计算机硬件类课程，系统集成了 Logisim 渲染引擎。
*   **功能**：上传 `.circ` 格式的 Logisim 电路文件后，系统会自动在后端进行无头（Headless）渲染。
*   **展示**：用户点击文件，即可在网页端查看生成的电路图高清预览，无需本地安装 Logisim 软件。
